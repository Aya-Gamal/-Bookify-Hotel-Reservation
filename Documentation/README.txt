Project Proposal:

Project Name:
Bookify Hotel Reservation System

Project Description:
Bookify is a full-stack hotel reservation web application built using ASP.NET Core MVC.
The system allows users to browse available hotel rooms, view room details, and make
reservations with an integrated online payment system.
It also provides an admin dashboard for managing rooms, room types, and reservations.

--------------------------------------------

User Roles:

✅ Admin User
- Manages room types
- Manages rooms
- Controls and monitors reservations

✅ Normal User (Customer)
- Browses available rooms
- Makes reservations
- Views booking history
